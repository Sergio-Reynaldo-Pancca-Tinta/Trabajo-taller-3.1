Explorando el Álgebra Relacional con Entity Framework

Implementación de operaciones con ADO.NET Entity Framework

HUAMAN SANCHEZ RIVALDO
PANCCA TINTA SERGIO REYNALDO

Introducción

Implementación de operaciones básicas del álgebra relacional (Selección, Proyección y Renombramiento) utilizando ADO.NET Entity Framework y la base de datos PUBS.

Objetivo: Demostrar la manipulación declarativa de datos mediante consultas LINQ.

Modelo de Datos

Se utilizó el modelo PUBS provisto por SQL Server. El diagrama fue generado mediante el archivo pubs.edmx.

Operaciones Implementadas

σ Selección

Filtrado de filas según condiciones específicas:

  • 5 consultas con diferentes tipos de predicados
  • Condiciones de comparación, rango y pertenencia
  • Filtrado de entidades específicas
// Ejemplo: Autores de California
var autoresCA = context.authors
    .Where(a => a.state == "CA");

π Proyección

Extracción de columnas específicas:

  • 5 consultas seleccionando subconjuntos de atributos
  • Eliminación de duplicados cuando es necesario
  • Selección de columnas relevantes
// Ejemplo: Solo nombres y apellidos
var nombres = context.authors
    .Select(a => new { a.au_fname, a.au_lname });

ρ Renombramiento

Cambio de nombre de atributos para mejorar claridad:

  • 5 consultas con atributos renombrados
  • Facilita la integración con otras operaciones
  • Mejora la legibilidad de los resultados
// Ejemplo: Renombrando columnas
var resultado = context.authors
    .Select(a => new {
        Nombre = a.au_fname,
        Apellido = a.au_lname
    });

Conclusión

La práctica permitió comprender cómo Entity Framework facilita la implementación de consultas inspiradas en el álgebra relacional utilizando sentencias LINQ. Además, se reforzó el uso del modelo edmx y el patrón de acceso a datos, demostrando la aplicabilidad de los conceptos teóricos en un entorno de desarrollo moderno.
